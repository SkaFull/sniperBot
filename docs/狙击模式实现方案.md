## 概述

### 什么是狙击模式？

狙击模式是 Solana 交易系统中的一种极速交易策略，通过**直接向当前 Leader 的 TPU 端口发送交易**，绕过传统的 RPC 路由，实现毫秒级交易确认。

### 交易传播路径对比

```
┌─────────────────────────────────────────────────────────────────────┐
│                        传统路径 (慢)                                 │
├─────────────────────────────────────────────────────────────────────┤
│  机器人 → RPC(HTTP) → Gossip广播 → 多节点转发 → Leader              │
│  延迟: 200-500ms                                                    │
│  问题: 经过多次网络跳转，容易被 MEV 机器人夹击                        │
└─────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────┐
│                        狙击路径 (快)                                 │
├─────────────────────────────────────────────────────────────────────┤
│  机器人 → Leader TPU(UDP) → 直接打包                                │
│  延迟: 10-50ms                                                      │
│  优势: 最少网络跳转，直接进入 Leader 交易池                           │
└─────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────┐
│                      Jito 路径 (最快+安全)                           │
├─────────────────────────────────────────────────────────────────────┤
│  机器人 → Jito Block Engine → Jito Validator                        │
│  延迟: 10-30ms                                                      │
│  优势: 原子性执行 + 防夹子 + 失败不付费                               │
└─────────────────────────────────────────────────────────────────────┘
```

### 核心目标

1. **计算当前 Leader 的 IP 地址**
2. **计算 Leader 的 TPU 端口**
3. **直接通过 UDP 发送交易到 Leader**

### 2.2 Leader 调度周期

```
┌────────────────────────────────────────────────────────────────────┐
│                      Solana Leader 调度周期                         │
└────────────────────────────────────────────────────────────────────┘

Epoch (纪元) ≈ 2-3 天
    │
    ├── 包含约 432,000 个 Slot
    │
    └── 每个 Slot = 400ms
        │
        ├── 每个 Slot 有一个指定的 Leader
        │
        └── Leader Schedule 在 Epoch 开始前就已确定

时间线:
┌─────────────────────────────────────────────────────────────────────┐
│ Epoch N-1          │ Epoch N            │ Epoch N+1               │
├────────────────────┼────────────────────┼─────────────────────────┤
│ ...Leader A...     │ ...Leader B...     │ ...Leader C...           │
│     Slot 100       │     Slot 200       │     Slot 300             │
│     (400ms)        │     (400ms)        │     (400ms)              │
└────────────────────┴────────────────────┴─────────────────────────┘

关键点:
1. Leader Schedule 是确定性的，可以提前计算
2. 当前 Epoch 的 Leader Schedule 在 Epoch 开始时就已经确定
3. 下一个 Epoch 的 Leader Schedule 在当前 Epoch 中期确定
```

### 3.1 Leader Schedule 计算公式

```typescript
/**
 * Leader Schedule 计算原理
 * 
 * Solana 使用确定性算法计算每个 Slot 的 Leader
 * 公式: leader_slot = slot % leader_schedule.num_slots
 */

interface LeaderSchedule {
  // 每个 Epoch 的 Slot 数量
  slotsPerEpoch: number;  // ≈ 432,000
  
  // Leader 排序表
  leaderSchedule: Uint8Array;  // 按顺序排列的 Leader 索引
  
  // 验证者列表
  validators: ValidatorInfo[];

  votePubkey: string;       // 投票账户公钥

}
```

### 3.3 计算当前 Slot 的 Leader

```typescript
/**
 * 计算当前 Slot 的 Leader
 */
async function getCurrentLeader(
  connection: Connection
): Promise<string> {
  // 1. 获取当前 Slot
  const currentSlot = await connection.getSlot();
  
  // 2. 获取当前 Epoch
  const epochInfo = await connection.getEpochInfo();
  const currentEpoch = epochInfo.epoch;
  
  // 3. 获取 Leader Schedule
  const schedule = await connection.getLeaderSchedule(currentEpoch);
  
  // 4. 找到当前 Slot 对应的 Leader
  const slotIndex = currentSlot % Object.values(schedule)[0].length;
  
  for (const [validator, slots] of Object.entries(schedule)) {
    if (slots.includes(slotIndex)) {
      return validator;
    }
  }
  
  throw new Error("Leader not found");
}
```

### 4.1 验证者身份解析流程

```

│                    Leader IP 地址计算流程                           │
└────────────────────────────────────────────────────────────────────┘

步骤 1: 获取当前 Slot
         │
         ▼
步骤 2: 获取 Leader Schedule
         │
         ▼

         │
         ▼
步骤 4: 通过 Vote Pubkey 获取 Identity Pubkey

         ▼

步骤 6: 解析出 IP 地址和端口
```

### 4.2 详细实现代码

```typescript
import { Connection, PublicKey } from "@solana/web3.js";

/**
 * Leader 信息结构
 */
interface LeaderInfo {
  identityPubkey: string;    // 验证者身份公钥
  votePubkey: string;        // 投票账户公钥
  ip: string;                // IP 地址
  tpu: number;               // TPU 端口
  tpuQuic: number;           // TPU QUIC 端口
  gossip: number;            // Gossip 端口
}

/**
 * 步骤 1: 获取当前 Slot 和 Epoch 信息
 */
async function getCurrentSlotInfo(connection: Connection): Promise<{
  slot: number;
  epoch: number;
  slotIndex: number;
}> {
  const epochInfo = await connection.getEpochInfo();
  
  return {
    slot: epochInfo.absoluteSlot,
    epoch: epochInfo.epoch,
    slotIndex: epochInfo.slotIndex
  };
}

/**
 * 步骤 2: 获取 Leader Schedule
 */
async function getLeaderSchedule(
  connection: Connection,
  epoch: number
): Promise<{ [key: string]: number[] }> {
  // 使用 'processed' 承诺级别获取最新数据
  const schedule = await connection.getLeaderSchedule(epoch, {
    commitment: 'processed'
  });
  
  if (!schedule) {
    throw new Error(`Failed to get leader schedule for epoch ${epoch}`);
  }
  
  return schedule;
}

/**
 * 步骤 3: 找到当前 Slot 的 Leader
 */
function findLeaderForSlot(
  schedule: { [key: string]: number[] },
  slotIndex: number
): string | null {
  for (const [validatorPubkey, slots] of Object.entries(schedule)) {
    if (slots.includes(slotIndex)) {
      return validatorPubkey;
    }
  }
  return null;
}

/**
 * 步骤 4: 获取验证者完整信息
 * 
 * 通过 RPC 的 getClusterNodes 方法获取所有验证者的网络信息
 */
async function getClusterNodes(connection: Connection): Promise<ClusterNode[]> {
  const nodes = await connection.getClusterNodes();
  return nodes;
}

interface ClusterNode {
  pubkey: string;      // 验证者公钥 (Gossip ID)
  gossip: string;     // Gossip 地址 (ip:port)
  tpu: string;        // TPU 地址 (ip:port)
  tpuQuic: string;    // TPU QUIC 地址 (ip:port)
  rpc: string | null; // RPC 地址 (可选)
  version: string;    // 节点版本
}

/**
 * 步骤 5: 匹配 Leader 和节点信息
 * 
 * 关键点: Leader Schedule 返回的是 Vote Pubkey
 *         而 getClusterNodes 返回的是 Identity Pubkey
 *         需要通过 Vote Accounts 进行映射
 */
async function getLeaderIdentity(
  connection: Connection,
  votePubkey: string
): Promise<string> {
  // 获取所有投票账户
  const voteAccounts = await connection.getVoteAccounts();
  
  // 在当前验证者中查找
  const currentValidator = voteAccounts.current.find(
    v => v.votePubkey === votePubkey
  );
  
  if (currentValidator) {
    return currentValidator.nodePubkey;  // Identity Pubkey
  }
  
  // 在待激活验证者中查找
  const delinquentValidator = voteAccounts.delinquent.find(
    v => v.votePubkey === votePubkey
  );
  
  if (delinquentValidator) {
    return delinquentValidator.nodePubkey;
  }
  
  throw new Error(`Identity not found for vote pubkey: ${votePubkey}`);
}

/**
 * 步骤 6: 解析 IP 和端口
 */
function parseAddress(address: string): { ip: string; port: number } {
  // 地址格式: "ip:port" 或 "[ipv6]:port"
  const lastColonIndex = address.lastIndexOf(':');
  
  if (lastColonIndex === -1) {
    throw new Error(`Invalid address format: ${address}`);
  }
  
  const ip = address.substring(0, lastColonIndex);
  const port = parseInt(address.substring(lastColonIndex + 1), 10);
  
  // 处理 IPv6 格式
  const cleanIp = ip.replace(/^\[|\]$/g, '');
  
  return { ip: cleanIp, port };
}

/**
 * 完整流程: 获取当前 Leader 的完整信息
 */
async function getCurrentLeaderInfo(connection: Connection): Promise<LeaderInfo> {
  console.log("=== 开始获取 Leader 信息 ===");
  
  // 1. 获取当前 Slot 信息
  console.log("步骤 1: 获取当前 Slot 信息...");
  const slotInfo = await getCurrentSlotInfo(connection);
  console.log(`  当前 Slot: ${slotInfo.slot}`);
  console.log(`  当前 Epoch: ${slotInfo.epoch}`);
  console.log(`  Slot Index: ${slotInfo.slotIndex}`);
  
  // 2. 获取 Leader Schedule
  console.log("步骤 2: 获取 Leader Schedule...");
  const schedule = await getLeaderSchedule(connection, slotInfo.epoch);
  console.log(`  获取到 ${Object.keys(schedule).length} 个验证者的调度信息`);
  
  // 3. 找到当前 Slot 的 Leader (Vote Pubkey)
  console.log("步骤 3: 查找当前 Slot 的 Leader...");
  const leaderVotePubkey = findLeaderForSlot(schedule, slotInfo.slotIndex);
  
  if (!leaderVotePubkey) {
    throw new Error("Leader not found for current slot");
  }
  console.log(`  Leader Vote Pubkey: ${leaderVotePubkey}`);
  
  // 4. 获取 Identity Pubkey
  console.log("步骤 4: 获取 Identity Pubkey...");
  const identityPubkey = await getLeaderIdentity(connection, leaderVotePubkey);
  console.log(`  Identity Pubkey: ${identityPubkey}`);
  
  // 5. 获取集群节点信息
  console.log("步骤 5: 获取集群节点信息...");
  const clusterNodes = await getClusterNodes(connection);
  console.log(`  获取到 ${clusterNodes.length} 个节点信息`);
  
  // 6. 匹配节点信息
  console.log("步骤 6: 匹配 Leader 节点信息...");
  const leaderNode = clusterNodes.find(node => node.pubkey === identityPubkey);
  
  if (!leaderNode) {
    throw new Error(`Node info not found for identity: ${identityPubkey}`);
  }
  
  // 7. 解析地址
  console.log("步骤 7: 解析网络地址...");
  const tpuAddr = parseAddress(leaderNode.tpu);
  const tpuQuicAddr = parseAddress(leaderNode.tpuQuic);
  const gossipAddr = parseAddress(leaderNode.gossip);
  
  const leaderInfo: LeaderInfo = {
    identityPubkey: identityPubkey,
    votePubkey: leaderVotePubkey,
    ip: tpuAddr.ip,
    tpu: tpuAddr.port,
    tpuQuic: tpuQuicAddr.port,
    gossip: gossipAddr.port
  };
  
  console.log("=== Leader 信息获取完成 ===");
  console.log(`  IP: ${leaderInfo.ip}`);
  console.log(`  TPU 端口: ${leaderInfo.tpu}`);
  console.log(`  TPU QUIC 端口: ${leaderInfo.tpuQuic}`);
  console.log(`  Gossip 端口: ${leaderInfo.gossip}`);
  
  return leaderInfo;
}
```

### 4.3 简化版实现 (推荐)

```typescript
/**
 * 简化版: 直接通过 getClusterNodes 获取所有节点
 *         然后通过 getSlotLeaders 获取当前 Leader
 */
async function getCurrentLeaderSimple(connection: Connection): Promise<LeaderInfo> {
  // 方法 1: 使用 getSlotLeaders (Solana Web3.js 2.0+)
  // 注意: 这个方法可能不是所有 RPC 都支持
  
  // 方法 2: 组合使用多个 API
  const [slotInfo, clusterNodes, voteAccounts] = await Promise.all([
    connection.getEpochInfo(),
    connection.getClusterNodes(),
    connection.getVoteAccounts()
  ]);
  
  // 获取 Leader Schedule
  const schedule = await connection.getLeaderSchedule(slotInfo.epoch);
  
  // 找到当前 Leader
  let leaderVotePubkey: string | null = null;
  for (const [pubkey, slots] of Object.entries(schedule || {})) {
    if (slots.includes(slotInfo.slotIndex)) {
      leaderVotePubkey = pubkey;
      break;
    }
  }
  
  if (!leaderVotePubkey) {
    throw new Error("Leader not found");
  }
  
  // 找到 Identity
  const validator = [...voteAccounts.current, ...voteAccounts.delinquent]
    .find(v => v.votePubkey === leaderVotePubkey);
  
  if (!validator) {
    throw new Error("Validator not found");
  }
  
  // 找到节点信息
  const node = clusterNodes.find(n => n.pubkey === validator.nodePubkey);
  
  if (!node) {
    throw new Error("Node not found");
  }
  
  // 解析 TPU 地址
  const tpuParts = node.tpu.split(':');
  const tpuQuicParts = node.tpuQuic.split(':');
  
  return {
    identityPubkey: validator.nodePubkey,
    votePubkey: leaderVotePubkey,
    ip: tpuParts[0],
    tpu: parseInt(tpuParts[1], 10),
    tpuQuic: parseInt(tpuQuicParts[1], 10),
    gossip: parseInt(node.gossip.split(':')[1], 10)
  };
}
```

## Leader TPU 端口计算

### 5.1 TPU 端口计算原理

```
┌────────────────────────────────────────────────────────────────────┐
│                      TPU 端口计算原理                               │
└────────────────────────────────────────────────────────────────────┘

Solana 验证者节点的端口分配规则:

基础端口 (Base Port): 由验证者配置决定
  - 默认基础端口: 8000 (可配置)
  - 通过 --dynamic-port-range 参数设置

端口偏移规则:
┌────────────────┬────────────────┬─────────────────────────────────┐
│ 服务名称        │ 偏移量          │ 计算公式                         │
├────────────────┼────────────────┼─────────────────────────────────┤
│ Gossip         │ +0             │ base_port + 0                   │
│ TPU            │ +1             │ base_port + 1                   │
│ TPU Quic       │ +2             │ base_port + 2                   │
│ TPU Forward    │ +3             │ base_port + 3                   │
│ TVU            │ +4             │ base_port + 4                   │
│ TVU Quic       │ +5             │ base_port + 5                   │
│ Repair         │ +6             │ base_port + 6                   │
│ Repair Quic    │ +7             │ base_port + 7                   │
│ RPC            │ +8 (可选)       │ base_port + 8                   │
│ RPC WebSocket  │ +9 (可选)       │ base_port + 9                   │
└────────────────┴────────────────┴─────────────────────────────────┘

示例:
如果 Gossip 端口是 8001，则:
  - TPU = 8002
  - TPU Quic = 8003
  - TVU = 8005
```

### 5.2 端口计算代码

```typescript
/**
 * 端口偏移量定义
 */
const PORT_OFFSETS = {
  GOSSIP: 0,
  TPU: 1,
  TPU_QUIC: 2,
  TPU_FORWARD: 3,
  TVU: 4,
  TVU_QUIC: 5,
  REPAIR: 6,
  REPAIR_QUIC: 7,
  RPC: 8,
  RPC_WEBSOCKET: 9
} as const;

/**
 * 根据已知端口计算其他端口
 */
function calculatePorts(knownPort: number, knownType: keyof typeof PORT_OFFSETS): {
  gossip: number;
  tpu: number;
  tpuQuic: number;
  tpuForward: number;
  tvu: number;
  tvuQuic: number;
  repair: number;
  repairQuic: number;
  rpc: number;
  rpcWebsocket: number;
} {
  // 计算基础端口
  const knownOffset = PORT_OFFSETS[knownType];
  const basePort = knownPort - knownOffset;
  
  return {
    gossip: basePort + PORT_OFFSETS.GOSSIP,
    tpu: basePort + PORT_OFFSETS.TPU,
    tpuQuic: basePort + PORT_OFFSETS.TPU_QUIC,
    tpuForward: basePort + PORT_OFFSETS.TPU_FORWARD,
    tvu: basePort + PORT_OFFSETS.TVU,
    tvuQuic: basePort + PORT_OFFSETS.TVU_QUIC,
    repair: basePort + PORT_OFFSETS.REPAIR,
    repairQuic: basePort + PORT_OFFSETS.REPAIR_QUIC,
    rpc: basePort + PORT_OFFSETS.RPC,
    rpcWebsocket: basePort + PORT_OFFSETS.RPC_WEBSOCKET
  };
}

/**
 * 从 Gossip 端口计算 TPU 端口
 */
function calculateTpuFromGossip(gossipPort: number): number {
  return gossipPort + 1;  // TPU = Gossip + 1
}

/**
 * 从 TPU 端口计算其他端口
 */
function calculateAllPortsFromTpu(tpuPort: number): {
  gossip: number;
  tpu: number;
  tpuQuic: number;
  tpuForward: number;
  tvu: number;
} {
  return {
    gossip: tpuPort - 1,
    tpu: tpuPort,
    tpuQuic: tpuPort + 1,
    tpuForward: tpuPort + 2,
    tvu: tpuPort + 3
  };
}
```

### 5.3 完整的 Leader 信息获取

```typescript
/**
 * 完整的 Leader 信息获取，包含端口计算
 */
async function getLeaderWithPorts(connection: Connection): Promise<{
  identity: string;
  vote: string;
  ip: string;
  ports: {
    gossip: number;
    tpu: number;
    tpuQuic: number;
    tpuForward: number;
    tvu: number;
    tvuQuic: number;
    repair: number;
  };
}> {
  // 获取集群节点信息
  const clusterNodes = await connection.getClusterNodes();
  
  // 获取当前 Epoch 信息
  const epochInfo = await connection.getEpochInfo();
  
  // 获取 Leader Schedule
  const schedule = await connection.getLeaderSchedule(epochInfo.epoch);
  
  if (!schedule) {
    throw new Error("Failed to get leader schedule");
  }
  
  // 找到当前 Leader
  let leaderVotePubkey: string | null = null;
  for (const [pubkey, slots] of Object.entries(schedule)) {
    if (slots.includes(epochInfo.slotIndex)) {
      leaderVotePubkey = pubkey;
      break;
    }
  }
  
  if (!leaderVotePubkey) {
    throw new Error("Leader not found");
  }
  
  // 获取投票账户信息
  const voteAccounts = await connection.getVoteAccounts();
  const allValidators = [...voteAccounts.current, ...voteAccounts.delinquent];
  
  const validator = allValidators.find(v => v.votePubkey === leaderVotePubkey);
  if (!validator) {
    throw new Error("Validator not found");
  }
  
  // 找到节点信息
  const node = clusterNodes.find(n => n.pubkey === validator.nodePubkey);
  if (!node) {
    throw new Error("Node not found");
  }
  
  // 解析地址
  const [gossipIp, gossipPort] = node.gossip.split(':');
  const [tpuIp, tpuPort] = node.tpu.split(':');
  const [tpuQuicIp, tpuQuicPort] = node.tpuQuic.split(':');
  
  // 验证 IP 一致性
  if (gossipIp !== tpuIp || tpuIp !== tpuQuicIp) {
    console.warn("Warning: IP addresses are inconsistent across services");
  }
  
  // 计算所有端口
  const ports = calculatePorts(parseInt(gossipPort, 10), 'GOSSIP');
  
  return {
    identity: validator.nodePubkey,
    vote: leaderVotePubkey,
    ip: gossipIp,
    ports: {
      gossip: ports.gossip,
      tpu: ports.tpu,
      tpuQuic: ports.tpuQuic,
      tpuForward: ports.tpuForward,
      tvu: ports.tvu,
      tvuQuic: ports.tvuQuic,
      repair: ports.repair
    }
  };
}
```

```typescript
// src/core/leader-tracker.ts

import { Connection } from "@solana/web3.js";
import { Logger } from "../utils/logger";
import { CacheManager } from "../utils/cache";

const logger = new Logger("LeaderTracker");
const cache = new CacheManager();

export interface LeaderInfo {
  identity: string;
  vote: string;
  ip: string;
  ports: {
    gossip: number;
    tpu: number;
    tpuQuic: number;
    tpuForward: number;
    tvu: number;
    tvuQuic: number;
    repair: number;
  };
  slot: number;
  epoch: number;
  expiresAt: number;  // 过期时间戳
}

export class LeaderTracker {
  private connection: Connection;
  private currentLeader: LeaderInfo | null = null;
  private updateInterval: NodeJS.Timeout | null = null;
  
  // 缓存配置
  private readonly CACHE_TTL = 1000 * 60 * 5;  // 5 分钟缓存
  private readonly UPDATE_INTERVAL = 1000 * 10;  // 10 秒更新间隔
  
  constructor(connection: Connection) {
    this.connection = connection;
  }
  
  /**
   * 启动 Leader 追踪
   */
  async start(): Promise<void> {
    logger.info("Starting Leader Tracker...");
    
    // 初始获取
    await this.updateLeader();
    
    // 定时更新
    this.updateInterval = setInterval(async () => {
      try {
        await this.updateLeader();
      } catch (error) {
        logger.error("Failed to update leader", error);
      }
    }, this.UPDATE_INTERVAL);
    
    logger.info("Leader Tracker started");
  }
  
  /**
   * 停止 Leader 追踪
   */
  stop(): void {
    if (this.updateInterval) {
      clearInterval(this.updateInterval);
      this.updateInterval = null;
    }
    logger.info("Leader Tracker stopped");
  }
  
  /**
   * 获取当前 Leader 信息
   */
  getCurrentLeader(): LeaderInfo | null {
    // 检查缓存是否过期
    if (this.currentLeader && Date.now() < this.currentLeader.expiresAt) {
      return this.currentLeader;
    }
    return null;
  }
  
  /**
   * 获取当前 Leader (强制刷新)
   */
  async getCurrentLeaderFresh(): Promise<LeaderInfo> {
    await this.updateLeader();
    if (!this.currentLeader) {
      throw new Error("Failed to get current leader");
    }
    return this.currentLeader;
  }
  
  /**
   * 获取接下来 N 个 Slot 的 Leader
   */
  async getNextLeaders(count: number): Promise<LeaderInfo[]> {
    const epochInfo = await this.connection.getEpochInfo();
    const schedule = await this.connection.getLeaderSchedule(epochInfo.epoch);
    
    if (!schedule) {
      throw new Error("Failed to get leader schedule");
    }
    
    // 构建完整的 Slot-Leader 映射
    const slotLeaderMap = this.buildSlotLeaderMap(schedule);
    
    const leaders: LeaderInfo[] = [];
    const clusterNodes = await this.connection.getClusterNodes();
    const voteAccounts = await this.connection.getVoteAccounts();
    
    for (let i = 0; i < count; i++) {
      const targetSlot = epochInfo.slotIndex + i;
      const leaderVotePubkey = slotLeaderMap.get(targetSlot);
      
      if (!leaderVotePubkey) continue;
      
      const validator = [...voteAccounts.current, ...voteAccounts.delinquent]
        .find(v => v.votePubkey === leaderVotePubkey);
      
      if (!validator) continue;
      
      const node = clusterNodes.find(n => n.pubkey === validator.nodePubkey);
      if (!node) continue;
      
      const [ip, gossipPort] = node.gossip.split(':');
      const ports = this.calculatePorts(parseInt(gossipPort, 10));
      
      leaders.push({
        identity: validator.nodePubkey,
        vote: leaderVotePubkey,
        ip,
        ports,
        slot: epochInfo.absoluteSlot + i,
        epoch: epochInfo.epoch,
        expiresAt: Date.now() + this.CACHE_TTL
      });
    }
    
    return leaders;
  }
  
  /**
   * 更新 Leader 信息
   */
  private async updateLeader(): Promise<void> {
    const startTime = Date.now();
    
    try {
      // 并行获取所有需要的数据
      const [epochInfo, clusterNodes, voteAccounts, schedule] = await Promise.all([
        this.connection.getEpochInfo({ commitment: 'processed' }),
        this.connection.getClusterNodes(),
        this.connection.getVoteAccounts(),
        this.connection.getLeaderSchedule(undefined, { commitment: 'processed' })
      ]);
      
      if (!schedule) {
        throw new Error("Failed to get leader schedule");
      }
      
      // 找到当前 Leader
      const leaderVotePubkey = this.findLeaderForSlot(schedule, epochInfo.slotIndex);
      
      if (!leaderVotePubkey) {
        throw new Error("Leader not found for current slot");
      }
      
      // 找到验证者信息
      const validator = [...voteAccounts.current, ...voteAccounts.delinquent]
        .find(v => v.votePubkey === leaderVotePubkey);
      
      if (!validator) {
        throw new Error(`Validator not found for vote pubkey: ${leaderVotePubkey}`);
      }
      
      // 找到节点信息
      const node = clusterNodes.find(n => n.pubkey === validator.nodePubkey);
      
      if (!node) {
        throw new Error(`Node not found for identity: ${validator.nodePubkey}`);
      }
      
      // 解析地址
      const [ip, gossipPort] = node.gossip.split(':');
      const ports = this.calculatePorts(parseInt(gossipPort, 10));
      
      // 更新缓存
      this.currentLeader = {
        identity: validator.nodePubkey,
        vote: leaderVotePubkey,
        ip,
        ports,
        slot: epochInfo.absoluteSlot,
        epoch: epochInfo.epoch,
        expiresAt: Date.now() + this.CACHE_TTL
      };
      
      const elapsed = Date.now() - startTime;
      logger.debug(`Leader updated in ${elapsed}ms`, {
        identity: this.currentLeader.identity,
        ip: this.currentLeader.ip,
        tpu: this.currentLeader.ports.tpu
      });
      
    } catch (error) {
      logger.error("Failed to update leader", error);
      throw error;
    }
  }
  
  /**
   * 查找指定 Slot 的 Leader
   */
  private findLeaderForSlot(
    schedule: { [key: string]: number[] },
    slotIndex: number
  ): string | null {
    for (const [pubkey, slots] of Object.entries(schedule)) {
      if (slots.includes(slotIndex)) {
        return pubkey;
      }
    }
    return null;
  }
  
  /**
   * 构建 Slot -> Leader 映射
   */
  private buildSlotLeaderMap(
    schedule: { [key: string]: number[] }
  ): Map<number, string> {
    const map = new Map<number, string>();
    
    for (const [pubkey, slots] of Object.entries(schedule)) {
      for (const slot of slots) {
        map.set(slot, pubkey);
      }
    }
    
    return map;
  }
  
  /**
   * 计算所有端口
   */
  private calculatePorts(gossipPort: number): LeaderInfo['ports'] {
    return {
      gossip: gossipPort,
      tpu: gossipPort + 1,
      tpuQuic: gossipPort + 2,
      tpuForward: gossipPort + 3,
      tvu: gossipPort + 4,
      tvuQuic: gossipPort + 5,
      repair: gossipPort + 6
    };
  }
}
```

```typescript
// src/core/tpu-sender.ts

import { VersionedTransaction } from "@solana/web3.js";
import dgram from "dgram";
import { Logger } from "../utils/logger";
import { LeaderInfo } from "./leader-tracker";

const logger = new Logger("TPUSender");

export interface TpuSendResult {
  success: boolean;
  leader: string;
  ip: string;
  port: number;
  error?: string;
  latency?: number;
}

export class TpuSender {
  private socket: dgram.Socket;
  private timeout: number;
  
  constructor(timeout: number = 5000) {
    this.timeout = timeout;
    this.socket = dgram.createSocket("udp4");
    
    // 错误处理
    this.socket.on("error", (err) => {
      logger.error("UDP socket error", err);
    });
  }
  
  /**
   * 直接发送交易到 Leader TPU
   */
  async sendTransaction(
    transaction: VersionedTransaction,
    leader: LeaderInfo
  ): Promise<TpuSendResult> {
    const startTime = Date.now();
    
    try {
      // 序列化交易
      const serializedTx = transaction.serialize();
      
      // 发送到 TPU 端口
      await this.sendUdp(
        serializedTx,
        leader.ip,
        leader.ports.tpu
      );
      
      const latency = Date.now() - startTime;
      
      logger.debug(`Transaction sent to Leader TPU`, {
        leader: leader.identity,
        ip: leader.ip,
        port: leader.ports.tpu,
        size: serializedTx.length,
        latency
      });
      
      return {
        success: true,
        leader: leader.identity,
        ip: leader.ip,
        port: leader.ports.tpu,
        latency
      };
      
    } catch (error) {
      const latency = Date.now() - startTime;
      
      logger.error("Failed to send transaction via TPU", {
        error,
        leader: leader.identity,
        ip: leader.ip,
        port: leader.ports.tpu
      });
      
      return {
        success: false,
        leader: leader.identity,
        ip: leader.ip,
        port: leader.ports.tpu,
        error: error instanceof Error ? error.message : String(error),
        latency
      };
    }
  }
  
  /**
   * 发送到 TPU QUIC 端口 (推荐)
   * 
   * 注意: QUIC 是 Solana 推荐的交易传输协议
   * 但需要更复杂的实现，这里提供 UDP 作为基础版本
   */
  async sendTransactionQuic(
    transaction: VersionedTransaction,
    leader: LeaderInfo
  ): Promise<TpuSendResult> {
    // TODO: 实现 QUIC 协议发送
    // 当前版本使用 UDP 作为替代
    return this.sendTransaction(transaction, {
      ...leader,
      ports: { ...leader.ports, tpu: leader.ports.tpuQuic }
    });
  }
  
  /**
   * 发送到多个 Leader (当前和下一个)
   */
  async sendToMultipleLeaders(
    transaction: VersionedTransaction,
    leaders: LeaderInfo[]
  ): Promise<TpuSendResult[]> {
    const results = await Promise.all(
      leaders.map(leader => this.sendTransaction(transaction, leader))
    );
    return results;
  }
  
  /**
   * UDP 发送
   */
  private sendUdp(data: Buffer, ip: string, port: number): Promise<void> {
    return new Promise((resolve, reject) => {
      const timeoutId = setTimeout(() => {
        reject(new Error("UDP send timeout"));
      }, this.timeout);
      
      this.socket.send(data, port, ip, (err) => {
        clearTimeout(timeoutId);
        if (err) {
          reject(err);
        } else {
          resolve();
        }
      });
    });
  }
  
  /**
   * 关闭 Socket
   */
  close(): void {
    this.socket.close();
    logger.info("UDP socket closed");
  }
}
```

### 6.4 Slot 监控器实现

```typescript
// src/core/slot-monitor.ts

import { Connection } from "@solana/web3.js";
import { Logger } from "../utils/logger";
import { LeaderTracker, LeaderInfo } from "./leader-tracker";

const logger = new Logger("SlotMonitor");

export interface SlotInfo {
  slot: number;
  epoch: number;
  slotIndex: number;
  leader: LeaderInfo | null;
  timestamp: number;
}

export type SlotCallback = (slotInfo: SlotInfo) => void;

export class SlotMonitor {
  private connection: Connection;
  private leaderTracker: LeaderTracker;
  private subscriptionId: number | null = null;
  private callbacks: SlotCallback[] = [];
  private lastSlot: number = 0;
  
  constructor(connection: Connection, leaderTracker: LeaderTracker) {
    this.connection = connection;
    this.leaderTracker = leaderTracker;
  }
  
  /**
   * 启动 Slot 监控
   */
  async start(): Promise<void> {
    logger.info("Starting Slot Monitor...");
    
    // 启动 Leader 追踪器
    await this.leaderTracker.start();
    
    // 订阅 Slot 更新
    this.subscriptionId = this.connection.onSlotChange(
      (slotInfo) => {
        this.handleSlotChange(slotInfo);
      }
    );
    
    logger.info("Slot Monitor started");
  }
  
  /**
   * 停止 Slot 监控
   */
  async stop(): Promise<void> {
    if (this.subscriptionId !== null) {
      await this.connection.removeSlotChangeListener(this.subscriptionId);
      this.subscriptionId = null;
    }
    
    this.leaderTracker.stop();
    logger.info("Slot Monitor stopped");
  }
  
  /**
   * 订阅 Slot 更新
   */
  onSlot(callback: SlotCallback): void {
    this.callbacks.push(callback);
  }
  
  /**
   * 处理 Slot 变化
   */
  private handleSlotChange(slotInfo: { slot: number; parent: number }): void {
    // 避免重复处理
    if (slotInfo.slot <= this.lastSlot) {
      return;
    }
    this.lastSlot = slotInfo.slot;
    
    // 获取当前 Leader
    const leader = this.leaderTracker.getCurrentLeader();
    
    const info: SlotInfo = {
      slot: slotInfo.slot,
      epoch: Math.floor(slotInfo.slot / 432000),  // 约每个 Epoch 432,000 slots
      slotIndex: slotInfo.slot % 432000,
      leader,
      timestamp: Date.now()
    };
    
    // 触发回调
    this.callbacks.forEach(callback => {
      try {
        callback(info);
      } catch (error) {
        logger.error("Callback error", error);
      }
    });
    
    logger.debug(`Slot ${slotInfo.slot}`, {
      leader: leader?.identity,
      ip: leader?.ip,
      tpu: leader?.ports.tpu
    });
  }
  
  /**
   * 获取当前 Slot 信息
   */
  async getCurrentSlot(): Promise<SlotInfo> {
    const epochInfo = await this.connection.getEpochInfo();
    const leader = this.leaderTracker.getCurrentLeader();
    
    return {
      slot: epochInfo.absoluteSlot,
      epoch: epochInfo.epoch,
      slotIndex: epochInfo.slotIndex,
      leader,
      timestamp: Date.now()
    };
  }
  
  /**
   * 等待下一个 Slot
   */
  async waitForNextSlot(): Promise<SlotInfo> {
    return new Promise((resolve) => {
      const callback = (slotInfo: SlotInfo) => {
        this.callbacks = this.callbacks.filter(cb => cb !== callback);
        resolve(slotInfo);
      };
      this.callbacks.push(callback);
    });
  }
}
```

### 6.5 完整的狙击交易发送器

```typescript
// src/core/sniper-sender.ts

import { 
  Connection, 
  VersionedTransaction,
  TransactionConfirmationStrategy
} from "@solana/web3.js";
import { Logger } from "../utils/logger";
import { LeaderTracker, LeaderInfo } from "./leader-tracker";
import { TpuSender } from "./tpu-sender";

const logger = new Logger("SniperSender");

export interface SniperConfig {
  // 是否使用 TPU 直连
  useTpuDirect: boolean;
  
  // 是否发送到多个 Leader
  sendToMultipleLeaders: boolean;
  
  // 预发送的 Leader 数量
  preSendLeaderCount: number;
  
  // 超时时间
  timeout: number;
  
  // 重试次数
  maxRetries: number;
}

export interface SendResult {
  success: boolean;
  signature?: string;
  method: "tpu" | "rpc" | "jito";
  leader?: string;
  latency?: number;
  error?: string;
}

export class SniperSender {
  private connection: Connection;
  private leaderTracker: LeaderTracker;
  private tpuSender: TpuSender;
  private config: SniperConfig;
  
  constructor(
    connection: Connection,
    leaderTracker: LeaderTracker,
    config: Partial<SniperConfig> = {}
  ) {
    this.connection = connection;
    this.leaderTracker = leaderTracker;
    this.tpuSender = new TpuSender(config.timeout || 5000);
    
    this.config = {
      useTpuDirect: true,
      sendToMultipleLeaders: true,
      preSendLeaderCount: 2,
      timeout: 5000,
      maxRetries: 3,
      ...config
    };
  }
  
  /**
   * 发送交易 (极速模式)
   * 
   * 策略:
   * 1. 首先尝试 TPU 直连当前 Leader
   * 2. 同时发送到下一个 Leader (防止 Leader 切换)
   * 3. 如果 TPU 失败，回退到 RPC
   */
  async sendTransaction(
    transaction: VersionedTransaction
  ): Promise<SendResult> {
    const startTime = Date.now();
    
    // 获取签名
    const signature = this.getTransactionSignature(transaction);
    
    try {
      // 策略 1: TPU 直连
      if (this.config.useTpuDirect) {
        const tpuResult = await this.sendViaTpu(transaction);
        
        if (tpuResult.success) {
          return {
            success: true,
            signature,
            method: "tpu",
            leader: tpuResult.leader,
            latency: Date.now() - startTime
          };
        }
      }
      
      // 策略 2: 回退到 RPC
      const rpcResult = await this.sendViaRpc(transaction);
      
      return {
        success: rpcResult,
        signature,
        method: "rpc",
        latency: Date.now() - startTime
      };
      
    } catch (error) {
      return {
        success: false,
        signature,
        method: "tpu",
        error: error instanceof Error ? error.message : String(error),
        latency: Date.now() - startTime
      };
    }
  }
  
  /**
   * 通过 TPU 发送
   */
  private async sendViaTpu(transaction: VersionedTransaction): Promise<{
    success: boolean;
    leader?: string;
  }> {
    try {
      // 获取当前 Leader
      const currentLeader = this.leaderTracker.getCurrentLeader();
      
      if (!currentLeader) {
        logger.warn("No leader info available, falling back to RPC");
        return { success: false };
      }
      
      // 发送到当前 Leader
      const leaders: LeaderInfo[] = [currentLeader];
      
      // 如果配置了多 Leader 发送
      if (this.config.sendToMultipleLeaders) {
        const nextLeaders = await this.leaderTracker.getNextLeaders(
          this.config.preSendLeaderCount
        );
        leaders.push(...nextLeaders.slice(0, this.config.preSendLeaderCount));
      }
      
      // 并行发送到所有 Leader
      const results = await this.tpuSender.sendToMultipleLeaders(
        transaction,
        leaders
      );
      
      // 只要有一个成功就认为成功
      const successResult = results.find(r => r.success);
      
      if (successResult) {
        logger.info("Transaction sent via TPU", {
          leader: successResult.leader,
          ip: successResult.ip,
          port: successResult.port,
          latency: successResult.latency
        });
        
        return { success: true, leader: successResult.leader };
      }
      
      return { success: false };
      
    } catch (error) {
      logger.error("TPU send failed", error);
      return { success: false };
    }
  }
  
  /**
   * 通过 RPC 发送
   */
  private async sendViaRpc(transaction: VersionedTransaction): Promise<boolean> {
    try {
      const signature = await this.connection.sendRawTransaction(
        transaction.serialize(),
        {
          skipPreflight: true,
          maxRetries: this.config.maxRetries
        }
      );
      
      logger.info("Transaction sent via RPC", { signature });
      return true;
      
    } catch (error) {
      logger.error("RPC send failed", error);
      return false;
    }
  }
  
  /**
   * 获取交易签名
   */
  private getTransactionSignature(transaction: VersionedTransaction): string {
    return Buffer.from(transaction.signatures[0]).toString("base64");
  }
  
  /**
   * 关闭资源
   */
  close(): void {
    this.tpuSender.close();
  }
}
```

### 7.1 缓存策略

```typescript
/**
 * Leader 信息缓存策略
 * 
 * 1. 缓存时间: 5 分钟 (Leader Schedule 在 Epoch 内不变)
 * 2. 预取策略: 提前获取下一个 Epoch 的 Leader Schedule
 * 3. 失效策略: Slot 变化时检查是否需要更新
 */

interface CacheConfig {
  // Leader 信息缓存 TTL
  leaderCacheTTL: number;  // 默认 5 分钟
  
  // 集群节点缓存 TTL
  clusterNodesCacheTTL: number;  // 默认 10 分钟
  
  // 投票账户缓存 TTL
  voteAccountsCacheTTL: number;  // 默认 5 分钟
  
  // 是否启用预取
  enablePrefetch: boolean;
  
  // 预取提前量 (Slot 数)
  prefetchSlots: number;  // 默认 10
}

class LeaderCache {
  private cache: Map<string, { data: any; expiresAt: number }> = new Map();
  
  get<T>(key: string): T | null {
    const cached = this.cache.get(key);
    if (cached && Date.now() < cached.expiresAt) {
      return cached.data as T;
    }
    this.cache.delete(key);
    return null;
  }
  
  set<T>(key: string, data: T, ttl: number): void {
    this.cache.set(key, {
      data,
      expiresAt: Date.now() + ttl
    });
  }
  
  invalidate(key: string): void {
    this.cache.delete(key);
  }
  
  clear(): void {
    this.cache.clear();
  }
}
```

### 7.3 预测性发送

```typescript
/**
 * 预测性发送策略
 * 
 * 在 Leader 切换前提前发送到下一个 Leader
 * 避免 Leader 切换时的延迟
 */
async function predictiveSend(
  transaction: VersionedTransaction,
  leaderTracker: LeaderTracker,
  tpuSender: TpuSender
): Promise<void> {
  // 获取当前和下一个 Leader
  const currentLeader = leaderTracker.getCurrentLeader();
  const nextLeaders = await leaderTracker.getNextLeaders(2);
  
  // 计算当前 Slot 剩余时间
  const slotRemainingMs = calculateSlotRemainingTime();
  
  // 如果剩余时间小于 100ms，同时发送到当前和下一个 Leader
  if (slotRemainingMs < 100) {
    const leaders = [currentLeader, ...nextLeaders].filter(Boolean) as LeaderInfo[];
    await tpuSender.sendToMultipleLeaders(transaction, leaders);
  } else {
    // 只发送到当前 Leader
    if (currentLeader) {
      await tpuSender.sendTransaction(transaction, currentLeader);
    }
  }
}

/**
 * 计算当前 Slot 剩余时间
 * Solana 每个 Slot 约 400ms
 */
function calculateSlotRemainingTime(): number {
  // 假设 Slot 开始时间为已知
  const slotStartTime = getSlotStartTime();  // 需要实现
  const slotDuration = 400;  // ms
  const elapsed = Date.now() - slotStartTime;
  return Math.max(0, slotDuration - elapsed);
}
```

```typescript
/**
 * UDP 连接池
 * 
 * 复用 UDP Socket，避免频繁创建销毁
 */
class UdpConnectionPool {
  private sockets: dgram.Socket[] = [];
  private currentIndex = 0;
  private poolSize: number;
  
  constructor(poolSize: number = 5) {
    this.poolSize = poolSize;
    this.initialize();
  }
  
  private initialize(): void {
    for (let i = 0; i < this.poolSize; i++) {
      const socket = dgram.createSocket("udp4");
      this.sockets.push(socket);
    }
  }
  
  getSocket(): dgram.Socket {
    const socket = this.sockets[this.currentIndex];
    this.currentIndex = (this.currentIndex + 1) % this.poolSize;
    return socket;
  }
  
  close(): void {
    this.sockets.forEach(socket => socket.close());
    this.sockets = [];
  }
}
```

## 总结

### 关键要点

1. **Leader IP 计算流程**:
   ```
   获取当前 Slot → 获取 Leader Schedule → 找到 Vote Pubkey
   → 通过 Vote Accounts 获取 Identity Pubkey
   → 通过 getClusterNodes 获取网络信息
   → 解析 IP 和端口
   ```

2. **TPU 端口计算规则**:
   ```
   TPU = Gossip Port + 1
   TPU QUIC = Gossip Port + 2
   TPU Forward = Gossip Port + 3
   ```

3. **性能优化关键**:
   - 并行请求所有需要的数据
   - 缓存 Leader 信息 (5 分钟 TTL)
   - 预测性发送到下一个 Leader
   - UDP 连接池复用

4. **最佳实践**:
   - 使用 `processed` 承诺级别获取最新数据
   - 同时发送到当前和下一个 Leader
   - 监控 Leader 切换时间点
   - 实现回退机制 (TPU → RPC)
